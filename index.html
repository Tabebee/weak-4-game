Crystal Collector

Rules

You will start out with a randomly generated number.

By now you should have noticed that there are 4 different colored crystals below. Each of those crystals has a different value assigned to it. Every time you click one of them the value of the crystal will be added to "Your Number."

Match "Your Number" with the "Target Number" and you Win! Go over the Target Number and you lose

The value for each crystal is a mystery for you...... until you click them!

If that wasn't enough well the value of each crystal will also change after a win or loss!
